# easy-flow-agent

OpenClaw improvements for the Easy Flow AI Agent Service.

## Packages

### workflow-controller
Workflow state management and step orchestration for multi-step AI agent workflows.

- Full state tracking (current step, completed steps, blockedReasons)
- Session persistence and recovery
- JSON serialization support
- UnifiedAgentState integration

## Development

```bash
npm install
npm test
npm run build
```
